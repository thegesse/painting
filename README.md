# Painting app in java

## About
A short project I made to pass the weekend, it is a small and simple drawing app made entirely in java. It isn't the most optimized yet, and there may be some bugs I haven't seen yet,
but It's my first time using canvas api in java so I am proud of myself.

## How to use
After downloading and opening it in a IDE of your choice simply run the launcher.java and have fun painting, you can also save your drawings as PNG with the save button.
